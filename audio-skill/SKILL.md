# Audio Skill — 音频规划提炼

用于从当前游戏组件中提取与音频系统相关的信息，统一输出 JSON，包含：
- BGM 曲目与用途
- 场景切换时的音频切换规则
- 自动播放失败后的兜底策略
- 音频生命周期与播放状态说明

## 适用场景

- 需要整理背景音乐配置表
- 需要明确不同状态下的切歌规则
- 需要输出给音频系统或播放器模块
- 需要排查浏览器自动播放与恢复播放策略

## 输出规范

推荐输出为 JSON 对象：

```json
{
  "audioType": "bgm",
  "summary": {},
  "playbackConfig": {},
  "tracks": [],
  "switchRules": [],
  "lifecycle": {},
  "autoplayFallback": {}
}
```

## 抽取规则

1. 曲目识别
- 提取所有背景音乐曲目、URL、默认曲目与适用场景

2. 切换规则
- 根据暂停、行动中、默认流程等状态整理优先级与目标曲目

3. 播放策略
- 提取循环、音量、恢复播放、切换模式等设置

4. 生命周期
- 提取挂载、切场景、卸载时的音频处理逻辑

## 结果要求

- 不混入角色台词与角色 TTS
- 保留状态条件与曲目的映射关系
- 允许附带 `notes`、`stateFlag`、`usage` 等扩展字段
