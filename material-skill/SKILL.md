# Material Skill — 素材规划提炼

用于从当前游戏组件中提取与角色无关的视觉素材信息，统一输出 JSON，包含：
- 场景步骤与关卡素材
- 交互道具与通用 UI
- 美术资源采购/制作清单
- 可直接用于文生图的提示词

## 适用场景

- 需要整理关卡所需图片、道具、UI 素材
- 需要给美术、外包或资源平台提供制作清单
- 需要为 AI 生成图片准备结构化 prompt
- 需要将“场景/道具资源”与“角色资源”分离

## 输出规范

推荐输出为以下几类 JSON：

```json
{
  "materialProfile": {},
  "artChecklist": {},
  "imagePrompts": {}
}
```

## 抽取规则

1. 场景与步骤
- 提取各关卡、各步骤的场景定义
- 区分全局场景与关卡专属场景

2. 道具与 UI
- 提取每关主任务道具、通用交互道具、通用 UI 容器
- 标注是否动态素材、是否需要多状态

3. 美术清单
- 为每个素材生成唯一 `materialId`
- 标注 `category`、`usage`、`spec`、`priority`

4. 文生图提示词
- 基于素材清单补充 `prompt`、`negativePrompt`、尺寸、透明背景要求
- 保持同一关卡步骤图的风格一致

## 结果要求

- 不混入角色形象、角色台词、角色状态
- 保留关卡/步骤/素材的对应关系
- 允许附带 `exportNotes`、`variants`、`deliveryRule` 等扩展字段
